# MoonLan — дорожная карта

Полный целевой набор функций (аналог LanTopoLog) и версии, в которых они появятся.

- [x] **v0.1** SNMP-опрос коммутаторов, BRIDGE-MIB, автоматическая топология, двухпанельный веб-интерфейс, поиск, демо-режим
- [ ] **v0.2** Ручное редактирование схемы, сохранение раскладки, настраиваемые контекстные меню устройств _(перенесён — сознательно пропущен, будет реализован в одной из следующих версий)_
- [x] **v0.3** Непрерывный Ping-мониторинг, дата/время последнего ответа, автообнаружение новых устройств, журнал новых MAC-адресов, определение IP и имён хостов (ARP/DNS)
- [x] **v0.4** Корректный алгоритм прямых связей (без ложных связей в звезде), LACP-агрегаты (IEEE8023-LAG-MIB), VLAN на портах (PVID и имена), обнаружение неуправляемых коммутаторов _(счётчики ifInErrors/ifOutErrors/ifInDiscards/ifOutDiscards перенесены в v0.5)_
- [ ] **v0.5** Тревоги: email, Telegram, Syslog; пороговые значения трафика; мониторинг трафика до порта коммутатора; счётчики ошибок портов (ifInErrors/ifOutErrors/ifInDiscards/ifOutDiscards)
- [ ] **v0.6** Контроль Spanning Tree (STP) и уведомления об изменениях топологии
- [ ] **v0.7** Экспорт схемы в PDF и Draw.io (diagrams.net); импорт дополнительных сведений по MAC-адресам
- [ ] **v0.8** Инвентаризация компьютеров через WMI/WinRM
